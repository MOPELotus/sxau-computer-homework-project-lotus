# 项目题目与术语更新说明

## 最新题目

课程项目题目已统一更新为：

`大数据与智能模型在跨境重大动物疫病预警中的应用分析`

## 与现有仓库的对应关系

当前 GitHub 仓库的核心流程保持不变，仍然覆盖：

- 多源资料归一化
- 非结构化材料解析
- 风险特征构建
- 预警评分与图表输出

因此，现有 `disease_intel` 代码目录、输出文件结构和运行命令可以继续沿用。

## 对外术语建议

在论文、PPT、封面、分工页、视频解说词中，建议统一按以下术语表述：

- `情报抽取` 统一写作 `大数据清洗与解析`
- `多特征融合` 统一写作 `多维预警指标构建`
- `LLM` 或 `大模型` 统一写作 `智能决策模型`

## 产出物引用建议

- `risk_trend.png` 可表述为 `预警趋势图`
- `processed_dataset.csv` 可表述为 `大数据清洗与解析结果表`
- `predictions.csv` 可表述为 `预警概率与风险等级结果表`
- `run_report.md` 可表述为 `阶段性预警分析简报`
